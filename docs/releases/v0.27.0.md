# JenAI v0.27.0 - TUI command queue and consistent transcript rhythm

## Added

- TUI submissions made while another command is active now enter a bounded FIFO queue instead of being rejected as busy.
- The status bar displays `queue N`; `/queue` lists pending items and `/queue clear` removes them.
- Approval cards pause queue consumption until the action is approved or rejected.
- Esc and `/abort` interrupt only the current command, then continue the queue.

## Safety

- `/stop` still preempts immediately and now clears every queued command, preventing stale movement intent from running after an emergency stop.
- The queue is capped at 20 commands and reports overflow instead of silently dropping entries.

## UI

- Prompt, assistant, tool, timeline, and approval entries use one consistent row gap.
- Normal wrapped lines remain adjacent; repeated model-authored paragraph gaps collapse to one blank row.

- `409 passed`; lint, safety coverage, build, and wheel smoke-test gates passed.
