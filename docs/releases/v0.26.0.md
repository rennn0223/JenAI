# JenAI v0.26.0 - Safe onboarding reset

## Added

- New `JenAI onboard` command to rerun the setup wizard at any time.
- Existing `config.toml` is copied to a timestamped `config.toml.bak-*` file before replacement.
- `.env`, locations, skills, reports, and run history are preserved.
- `--yes` / `-y` skips the replacement confirmation for scripted onboarding.

## Verification

- Integration coverage for backup/preservation, cancellation, and first-time onboarding.
- `404 passed`; lint, build, and wheel smoke-test gates passed.
