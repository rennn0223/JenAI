**補完只補指令名,格式變提示** —— 應使用者回饋:「Tab 補全只需要提示後面的格式,但不要真的把格式打出來——都要先刪掉再輸入很麻煩。」有道理,改。

## 改了什麼
- **TUI**:Tab 補完現在只插入「指令名 + 空格」,游標停在行尾直接打參數;`<placeholder>` 模板**不再**進輸入框。參數格式改為 **palette 淡色提示**(補完後 palette 留著顯示「格式 /ros pub `<topic> <payload>`」+ 說明,打參數時一直在、Esc 可關、不攔輸入)
- **WebUI**:同款行為——Tab/點擊補完後,面板轉為單行格式提示(非互動),打參數不被劫持,Esc 關閉
- 佔位符防呆網保留:手打/複製貼上的 `<placeholder>` 送出仍會被擋(它曾把字面 `<name|number>` 存成 model binding)
- 順手修:`render.py` 的 `_PAGE` 改 raw string,消除 JS regex `\s` 的 SyntaxWarning

## 驗證
- TUI 測試改寫:補完=名稱+空格、palette 轉提示、續打參數提示仍在(**378 total** 全綠);ruff 全綠
