**`/mode` —— Shift+Tab 的鍵盤備援** —— 應使用者回饋:「他的 Shift+Tab 我按不了。」很多終端(SSH client、內嵌終端)根本不把 Shift+Tab 當獨立按鍵送出,權限模式不能被一顆按不到的鍵綁架。

## 改了什麼
- **新增 `/mode` slash 指令**:不帶參數 = 循環切換(同 Shift+Tab);`/mode approve|plan|auto` 直接指定,中文別名(審批/規劃/自動)也通;錯誤參數不改變模式並提示用法
- 與 Shift+Tab 走**同一條路**(`_set_mode`):狀態列 chip、時間軸切換紀錄(自動模式的可稽核性)完全一致,無第二套狀態
- `/help` 補 System 群組條目與 Shift+Tab 快捷鍵說明;COMMANDS/TECHNICAL_GUIDE 同步
