# JenAI v0.34.0 - 地點管理補全:/loc move · rename · rm

**應使用者實測需求(Carter 場景建點後想改名/改座標/刪點):`/loc` 從只能加,補成可管理。**

## 新功能

- **`/loc move <名>`**:把既有地點更新為機器人目前位置——「改座標」的實務形態就是重新站位再存;tags/aliases 全保留
- **`/loc rename <舊> <新>`**:改名保留座標;與其他名稱/alias 相撞誠實拒絕;名稱含空白用 `舊名 -> 新名` 消歧
- **`/loc rm <名>`**:刪除地點並回報被刪座標
- **安全語意**:三個操作一律**精確名稱比對**(不分大小寫)——alias 與模糊比對不觸發破壞性操作,找不到時誠實列出已知名稱;palette、`/help`、COMMANDS.md 同步
