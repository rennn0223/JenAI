**對話行距固定加大** —— 應使用者回饋:「希望對話(我的問題跟 LLM 的答案)的行距可以大一點且固定,像 Claude Code。」

## 改了什麼
- **答案(`Result`)改走 airy 節奏**:v0.22 純聊天串流換成 run agent 後,LLM 答案由 `OutputPanel` 緊排渲染,失去了舊聊天的呼吸感。現在 `OutputPanel` 支援 `spaced=True`——邏輯行之間**恰好一行**空行,模型自己亂空的多重空行會被收斂,行距因此「大且固定」
- **提問回顯(`>` 行)上下各固定一行間距**(原本下方是 0,答案貼著問題)
- 範圍收斂:只有 LLM 答案走 spaced;topics 表格、status、help 等列表輸出維持緊排

## 驗證
- 新增 OutputPanel spaced 節奏測試(**376 total**);ruff 全綠
- 實際渲染對照:`第一段\n\n\n第二段` → 段落間恰好一行空行;Status 表格不受影響
