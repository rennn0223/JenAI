- **E2 Twin Gate 消融**(N=100,每類 20):硬陷阱三類(界外/不可達/禁區內)**SIR 60/60 = 100%**;良性對照 **FPR 0/20 = 0%**;zone_crossing 由預演軌跡裁定(block 6/refer 2/pass 12,pass 均為 Nav2 實際繞行)

## 本版變更(自 v0.37.0)

- **doctor 不再對關閉的 Twin Gate 靜默**:`[twin] enabled = false` 時顯式 WARN
  「Twin Gate is DISABLED — goals go straight to the robot」——B7 首輪排練實證
  「安靜的關閘 + 全綠」會被讀成「閘驗過了」,誠實回報原則不容許這種靜默
- `/model` 箭頭選單(`ModelPicker`:↑/↓ 移動、Enter 選、Esc 取消、數字快跳;端點列不出時退回編號清單)

- 測試收斂:`/model` 測項統一於 test_tui_model_command.py、twin doctor 測項統一於 test_twin_gate.py(471 項全綠)
