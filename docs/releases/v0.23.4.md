**WebUI slash 指令選擇表** —— 應使用者回饋:「WebUI 沒有 slash 指令選擇表。」現在有了,而且跟 TUI 同款。

## 改了什麼
- **Console 輸入 `/` 彈出指令面板**:即時前綴過濾(打 `/ros ` 只剩六條子指令)、↑↓ 選擇(❯ 標記)、Tab 或點擊補完、Esc 關閉;打純文字或開始輸入參數時自動讓路
- **清單與實作同源**:面板資料 = `webui/commands.py` 的 `WEB_SLASH_COMMANDS`,server 端渲染進頁面;單元測試強制「每一條都必須對應 `_slash` 真實 handler」——幻影指令(v0.23.0 清掉的那種)從此進不了面板
- 樣式沿用儀表板設計語言(卡片、圓角、accent 選中底色),手機上點選同樣可用

## 誠實回答順帶的問題
**WebUI 會顯示 tool calling 嗎?不會。**WebUI Console 的純文字走單輪聊天(`ask_provider`),不跑 agent,所以沒有工具呼叫可顯示;TUI 才有完整的 agent 管線(`⏺ 工具 · 參數` + `⎿ 結果`)。「WebUI 跑 agent + 工具呼叫時間軸」是一個真功能(需要 run 狀態流式推送 + 批准卡整合),已列入 ROADMAP 軌道 5 候選,不假裝這次做了。
