# JenAI v0.36.1 - WebUI 地圖與 topics 清單改版(手機實測回饋)

**使用者手機開 WebUI 的三條回饋:地圖擠成一團、沒有縮放、110 個 topic 一大半標 (unknown)。**

## 地圖(Console 頁)

- **縮放/平移**:右上 + / − / ⤢(fit)按鈕、滑鼠滾輪縮放、拖曳平移(手機單指拖);第一次放大自動以機器人為中心
- **標籤防重疊**:貪婪去重——會蓋到別人的標籤自動隱藏(點保留、hover 有 tooltip),meta 列提示「N labels hidden (zoom in)」;放大後自然逐一浮現
- **格線降密**:跨度大時自動改 2/5/10…公尺一格,不再糊成灰帶
- 擠成一團的主因其實是兩個地圖外的 GPS 舊點(機械系館在 (88,-184),自動縮放為了裝下它把整個倉庫壓成幾個像素)——已依使用者指示刪除,`inspect` 技能同步改為倉庫四點巡檢

## Topics 清單(Status 頁 chips + API 頁)

- **分類器大幅擴充**(TUI `/ros topics` 同享):新增 `nav`(costmap/plan/amcl/goal…)、`tf`、`infra`(transition_event/bond/rosout/lifecycle…)——Carter+Nav2 的 110 個 topic 從「大半 unknown」變成 nav 54 / sensor 29 / infra 19 / control 3 / tf 2 / unknown 2
- **infra 收摺**:兩處清單都把 lifecycle/bond/rosout 等管線 topic 摺進 `details`(顯示數量),110 個 topic 讀起來像重要的那 ~30 個;110 這個數字本身是正常的(Nav2 全家桶 + 模擬器),不是 bug

## 驗證

- 分類器單元測試更新(含「amcl 的 transition_event 屬 infra 不屬 nav」的優先序);全測試綠、ruff 綠
- 對活 WebUI 實測:map payload 只剩倉庫 5 點、kind 統計如上、頁面含縮放鈕與收摺
