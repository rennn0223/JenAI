# JenAI v0.31.0 - 方向定調:AI Decision Agent,不是 AI Control Agent

**方向級變更(文件),兩個實證改寫投資方向:Isaac 孿生實測證明單 depth 反應式避障不可行;兩台載具皆自帶 SLAM+Nav。結論:JenAI 不寫一行運動控制。**

## 定位最終形態
> JenAI = **AI Decision Agent + 安全閘層**,坐在任何載具原生導航堆疊之上——自然語言/自主決策 → 封閉動作集 → 批准 + Twin 預演 → goal 交給載具自己的 nav → 監督、回饋、稽核。

## 隨之而來的決定
- **內建避障(stop-and-go detour)與 odom 直驅 → deprecated**:降為 bring-up/除錯的誠實 fallback,只修 bug、不再投資能力(程式碼保留——安全鏈行為不倒退)
- **軌道 2「感知深化」改向「餵腦不餵輪」**:VLM 語意進決策快照與 Gate 判準;幾何避障深化正式判死,負面結果寫入論文第五章失效分析
- **軌道 3 / B1 改寫**:從「自建圖+Nav2 bringup」變成「原生 nav 接口確認」——車邊各跑一次 `ros2 action list | grep -i navigate`;有 `NavigateToPose` 即直通,否則層一寫薄 adapter(接線,非控制)
- **M6 決策迴圈升為唯一程式主軸**;`/drive`、`/ros pub` 定位為工程師除錯工具
